## Zotero Tips

To copy-paste a citation from Zotero into references.bib, 

1. Zotero Windows > Edit > Preferences > Export > Item Format > BibTeX  
2. Select the reference  
3. Edit > Copy as BibTeX  
4. reference.bib > Paste
